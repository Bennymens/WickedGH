# Wicked Gh — React eCommerce Starter

Minimal Vite + React starter for the Wicked Gh e-commerce site.

Getting started

1. Install dependencies:

```powershell
npm install
```

2. Run development server:

```powershell
npm run dev
```

Build:

```powershell
npm run build
```

Files

- `package.json`: project scripts and deps
- `index.html`, `src/` — app source
